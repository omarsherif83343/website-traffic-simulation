# Website Traffic Simulation

This project simulates website traffic using Python and NumPy.

## Features
- Poisson distribution to simulate visitors per hour
- Statistical analysis (max, min, average)
- Array reshaping and slicing
- Binomial distribution to simulate user registrations
- Normal distribution to simulate session duration
- Data visualization using Matplotlib

## Technologies Used
- Python
- NumPy
- Matplotlib
